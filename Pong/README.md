# Pong Game

A Pong Game developed for learning purposes, using Allegro5.
If you have any question, please, send me.

### Prerequisites
```
C++ 11+
Allegro5
```

## Built With

* Allegro5 - Primitives Addon
* Allegro5 - Audio Addon
* Allegro5 - Audio Codec Addon
* Allegro5 - Fon Addon
